## What this video shows

How to install the playback-runner agent and use it to ask questions about Playback.

## What you will see

1. Set up a temp project directory
2. Run `playback init-agent` to install the agent files
3. Confirm two agent files are created in the project
4. Ask "How do I work this?" and see the agent respond
5. Ask "Can I make a playlist?" and see what it knows
6. Ask "Where do my tapes end up?" and get a clear answer
7. Closing narration

## What you will need

- `playback-cli` installed globally (`npm install -g @philsherry/playback-cli`)
- Claude Code or GitHub Copilot (to run the agent)

## What comes next

Try writing your first tape, or open the TUI timing editor to fine-tune your narration timing.
